Coastal Data

This is a prototype of a collaborative curation service for Coastal Data.

About the Project

Find Coastal Data

Browse or search the data blog to find coastal data of interest. Each entry discusses previous data uses and existing software.

Discuss Coastal Data

Use the commenting feature for each data blog post to discuss the coastal data sets and their use.

Collaborative coastal data science

You can add data blog entries to expand the collection. Discussions are intended to also spark collaborations with other data scientists and researchers.
